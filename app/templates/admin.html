Admin tools

The live chat experiment has been retired so we can focus on bookings, enquiries, and our core walking schedule. Use the forms below to stay on top of day-to-day admin.

Booking schedule

Manage availability through the public booking page. Slots that you add via the schedule API instantly appear for customers, and you can continue to confirm or cancel requests from your email notifications.

Endpoint: POST /api/admin/slots for new slots and PATCH /api/admin/bookings/<id>/status to confirm visits.

Contact enquiries

Every submission from the contact page is logged in memory. Fetch them via GET /api/admin/enquiries and mark enquiries as completed once you have replied so the counts remain accurate.

Visitor bans

Need to block a repeat spammer? Use the banned visitor endpoints to add, unban, or delete restrictions. The middleware still respects these records even without live chat.

Endpoint: POST /api/admin/banned-visitors
